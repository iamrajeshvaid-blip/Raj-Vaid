```html

<script async src="https://www.googletagmanager.com/gtag/js?id=GA_MEASUREMENT_ID"></script>

  window.dataLayer = window.dataLayer || [];
```

## TROUBLESHOOTING POST-DEPLOYMENT

### App won't load
1. Check browser console (F12)
2. Verify all files uploaded
3. Check manifest.json syntax (JSON validator tool)
4. Try hard refresh (Ctrl+Shift+R)

### Service worker not working
1. DevTools â†’ Application â†’ Service Workers
2. Unregister old worker
3. Hard refresh
4. Check service-worker.js for errors

### PWA not installing
1. Ensure HTTPS (production only)
2. Check manifest.json completeness
3. Verify icons path (must be absolute or relative)
4. Test on mobile (not desktop browser)

### White screen on mobile
1. Check phone's browser compatibility
2. Clear app cache
3. Try Safari (iOS) or Chrome (Android)
4. Check manifest.json
